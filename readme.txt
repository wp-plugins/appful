== Installation ==

Setting up a mobile App with appful is surprisingly simple - trust us!

1. Sign up at [appful.net](https://appful.net/ "Create mobile iOS Blog Apps")
2. Create your first app
3. Download and install the Plugin
4. Move to the "appful" section in the Wordpress menu
5. There enter your appful user information from the account you previously created
6. Navigate in the [my.appful](https://my.appful.net/ "appful Dashboard") Dashboard to INFO > Connect and activate your blog
